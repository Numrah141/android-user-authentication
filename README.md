# android-user-authentication
A simple Android application built in Java using SQLite database that implements user registration, login, password update, and account deletion functionality.

# Android User Authentication App

This project is a basic Android application developed using **Java** and **SQLite**.  
It demonstrates a complete user authentication flow including:

- User Registration (Sign Up)
- User Login (Sign In)
- Forget Password
- Update Password
- Delete Account

The application is designed for academic purposes and follows clean and beginner-friendly coding practices.
